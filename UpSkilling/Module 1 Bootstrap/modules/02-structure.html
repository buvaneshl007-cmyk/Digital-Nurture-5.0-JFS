└── index.html
📁 CSS Folder - What's Inside?
bootstrap.min.css

Size: ~173 KB | Use: Production environments

Minified CSS file containing all Bootstrap styles. All unnecessary characters removed for faster loading.

bootstrap.css

Size: ~230 KB | Use: Development/Learning

Human-readable CSS file. Great for understanding how Bootstrap is built.

bootstrap.rtl.css

Purpose: Right-to-Left language support (Arabic, Hebrew)

Use this if you're building websites for RTL languages.

.map files

Purpose: Source maps for debugging in browser DevTools

Helps developers see original Bootstrap source instead of minified version when debugging.

📁 JS Folder - What's Inside?
bootstrap.bundle.min.js

Size: ~56 KB | Includes: Bootstrap + Popper.js

✅ RECOMMENDED! Use this file. It includes everything needed for Bootstrap plugins.

bootstrap.bundle.js

Same as above but unminified, better for learning.

bootstrap.min.js

Size: ~30 KB | Includes: Bootstrap only

If you use this, you must separately include Popper.js library.

Exercise 2.2: Using JavaScript Plugins

Add Bootstrap's JavaScript plugins to enable interactive components

What Bootstrap JavaScript Enables
  • Dropdowns: Click to toggle dropdown menus
  • Modals: Pop-up dialogs
  • Navbars: Responsive mobile menu toggle
  • Tooltips & Popovers: Hover/click info boxes
  • Carousels: Image sliders
  • Collapse: Expand/collapse content
  • Tabs: Switchable tab panels
  • Scrollspy: Auto-highlight navigation items
Complete HTML Template with JavaScript
<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Bootstrap 5 with JavaScript</title> <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet"> </head> <body> <!-- Content Here --> <div class="container mt-5"> <h1>Bootstrap 5 with Interactive Features</h1> <!-- Modal Button --> <button class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#exampleModal"> Open Modal </button> <!-- Modal --> <div class="modal fade" id="exampleModal"> <div class="modal-dialog"> <div class="modal-content"> <div class="modal-header"> <h5 class="modal-title">Modal Title</h5> <button class="btn-close" data-bs-dismiss="modal"></button> </div> <div class="modal-body"> This is a Bootstrap modal! </div> </div> </div> </div> </div> <!-- IMPORTANT: JavaScript must be at bottom --> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script> </body> </html>
✅ Best Practices
  • ✅ Always use bootstrap.bundle.min.js (includes Popper.js)
  • ✅ Place script tag at the END of body tag (better loading)
  • ✅ Never put script in head section (blocks rendering)
  • ✅ Use data-bs-* attributes for interactive behavior
