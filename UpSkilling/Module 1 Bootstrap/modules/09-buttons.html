Exercise 9.2: Button Groups & Toggle Buttons

Create button groups and toggle buttons with checkboxes

Button Group Basics

Group multiple buttons together with .btn-group:

Button Group Code
<div class="btn-group" role="group"> <button type="button" class="btn btn-outline-primary">Left</button> <button type="button" class="btn btn-outline-primary">Middle</button> <button type="button" class="btn btn-outline-primary">Right</button> </div>
Toggle Buttons with Checkboxes

Create toggle buttons that can be activated/deactivated:

Toggle Button Code
<div class="btn-group" role="group"> <input type="checkbox" class="btn-check" id="check1" autocomplete="off"> <label class="btn btn-outline-primary" for="check1">Option 1</label> <input type="checkbox" class="btn-check" id="check2" autocomplete="off"> <label class="btn btn-outline-primary" for="check2">Option 2</label> </div>
