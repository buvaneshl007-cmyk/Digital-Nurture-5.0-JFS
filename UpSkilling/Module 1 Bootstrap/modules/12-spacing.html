Exercise 12.1: Margin & Padding

Apply spacing utilities to layout sections

Spacing Classes
Class Property Values
.m-1 to .m-5 Margin all sides 0.25rem to 3rem
.p-1 to .p-5 Padding all sides 0.25rem to 3rem
.mt-3 Margin-top Top margin only
.mb-3 Margin-bottom Bottom margin only
.ms-3 Margin-start (left) Left margin
.me-3 Margin-end (right) Right margin
.mx-3 Margin horizontal Left & right
.py-3 Padding vertical Top & bottom
Spacing Scale (1-5)
  • .m-1 = 0.25rem (4px)
  • .m-2 = 0.5rem (8px)
  • .m-3 = 1rem (16px)
  • .m-4 = 1.5rem (24px)
  • .m-5 = 3rem (48px)
Visual Demo

Responsive Spacing
