Exercise 16.1: Fixed Footer

Create a fixed footer using position utilities

Position Classes
  • .position-static - Default positioning
  • .position-relative - Relative to normal position
  • .position-absolute - Absolute within positioned parent
  • .position-fixed - Fixed to viewport
  • .position-sticky - Sticky positioning
Fixed Footer Code

Add to your HTML body (after main content):

<footer class="position-fixed bottom-0 start-0 end-0 bg-dark text-white p-3">
    <div class="container text-center">
        <p>© 2024 My Website. All rights reserved.</p>
    </div>
</footer>

<!-- Add padding to body to make room -->
<body style="padding-bottom: 100px;">
Your Task
  1. Create exercise16-1.html
  2. Create a page with:
    • Long scrollable content
    • Fixed footer at bottom
    • Proper spacing (padding-bottom)
