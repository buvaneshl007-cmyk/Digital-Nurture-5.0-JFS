Exercise 19.1: Setup Bootstrap with Sass

Set up a Bootstrap 5 project with Sass (No Command Line)

What is Sass?

Sass (Syntactically Awesome Stylesheets) is a CSS preprocessor that allows you to:

  • Use variables for colors, sizes, fonts
  • Use mixins to reuse code blocks
  • Nest selectors for better organization
  • Customize Bootstrap easily
Simple Approach: Override with CSS

For beginners, you can customize Bootstrap with regular CSS:

<!-- Link Bootstrap CSS first --> <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet"> <!-- Then your custom CSS (overrides Bootstrap) --> <style> :root { --bs-primary: #667eea; --bs-secondary: #764ba2; --bs-success: #00d084; } .btn-primary { border-radius: 50px; padding: 12px 30px; } </style>
Bootstrap CSS Variables
Variable Purpose
--bs-primary Primary color
--bs-secondary Secondary color
--bs-success Success color
--bs-danger Danger color
--bs-body-font-size Base font size
--bs-border-radius Border radius

Exercise 19.2: Advanced Customization

Learn advanced Bootstrap customization techniques

Custom Theme with CSS Variables
<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Custom Bootstrap Theme</title> <!-- Bootstrap CSS --> <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet"> <style> :root { --bs-primary: #7c3aed; --bs-secondary: #06b6d4; --bs-success: #10b981; --bs-danger: #ef4444; --bs-warning: #f59e0b; --bs-info: #3b82f6; --bs-body-font-family: "Segoe UI", Roboto, sans-serif; --bs-border-radius: 8px; --bs-border-radius-lg: 12px; } /* Custom button style */ .btn { font-weight: 600; letter-spacing: 0.5px; } /* Custom card style */ .card { border: none; box-shadow: 0 2px 8px rgba(0,0,0,0.1); } </style> </head> <body> <!-- Your content --> </body> </html>
Customization Tips
🎨 Easy Ways to Customize:
  • Use CSS custom properties (--bs-* variables)
  • Add your own CSS after Bootstrap
  • Override specific classes
  • Create a custom theme CSS file
For Advanced Users: Full Sass Setup

To use full Sass customization (requires development setup):

  1. Install Node.js and npm
  2. Install Sass: npm install sass
  3. Download Bootstrap source
  4. Create custom _variables.scss
  5. Compile with Sass compiler
Note: Full Sass customization requires command-line knowledge. The CSS variable approach is simpler for beginners!
