Exercise 11.1: Profile Card

Create a profile card using card components

Card Structure
  • .card - Main container
  • .card-img-top - Image at top
  • .card-body - Content area
  • .card-title - Title
  • .card-text - Body text
  • .card-footer - Bottom area
Profile Card Code
<div class="card" style="width: 300px;"> <img src="image.jpg" class="card-img-top" alt="Profile"> <div class="card-body text-center"> <h5 class="card-title">John Doe</h5> <p class="card-text text-muted">Web Developer</p> <p class="card-text">Passionate about creating amazing web experiences.</p> <div class="d-grid gap-2"> <button class="btn btn-primary">Follow</button> <button class="btn btn-outline-primary">Message</button> </div> </div> <div class="card-footer bg-light text-center text-muted"> <small>Member since 2023</small> </div> </div>

Exercise 11.2: Media Object Layout

Design a media object layout with image and content

Media Object Pattern

Combine image with text/content aligned horizontally:

<div class="media"> <img src="avatar.jpg" class="me-3" style="width: 64px;"> <div class="media-body"> <h5>Media heading</h5> Some content goes here... </div> </div>
