Exercise 5.1: Content Centering

Use flexbox utilities to center content horizontally and vertically

Alignment Utilities
Class Effect
.justify-content-center Center columns horizontally in row
.justify-content-between Space between columns (left & right edges)
.justify-content-around Equal space around each column
.align-items-center Center columns vertically in row
.align-items-start Align to top
Horizontal Centering Example
<div class="row justify-content-center"> <div class="col-md-6"> <div class="card"> <div class="card-body"> <h5>Centered Column</h5> <p>This column is centered on the page</p> </div> </div> </div> </div>

Vertical Centering Example
<div class="row align-items-center" style="min-height: 300px;"> <div class="col-md-6"> <p>This content is centered vertically</p> </div> </div>

Exercise 5.2: Column Reordering

Reorder columns on different screen sizes using order utilities

Order Classes

Bootstrap provides order utilities to change visual order without changing HTML:

.order-1 to .order-5 - Sets column order
.order-md-2 - Change order starting at medium breakpoint
.order-lg-1 - Change order starting at large breakpoint
Code Example
<div class="row"> <!-- Desktop: 1st | Mobile: 2nd --> <div class="col-md-8 order-md-1 order-2"> <p>Main Content</p> </div> <!-- Desktop: 2nd | Mobile: 1st --> <div class="col-md-4 order-md-2 order-1"> <p>Sidebar</p> </div> </div>

Real-World Use Cases
  • Mobile: Show sidebar first (important on small screens)
  • Desktop: Show main content first (better for reading flow)
  • Highlight important sections on mobile
  • Different content hierarchies for different devices
