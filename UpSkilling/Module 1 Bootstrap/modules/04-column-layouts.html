Exercise 4.1: Sidebar Layout

Create a two-column layout with sidebar (25%) and content area (75%)

Layout Structure

This is a common pattern for dashboards, blogs, and admin panels:

Header / Navbar
Sidebar (25%)
Main Content (75%)
Footer
Complete HTML Code
<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Exercise 4.1: Sidebar Layout</title> <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet"> </head> <body> <!-- Navbar --> <nav class="navbar navbar-expand-lg navbar-dark bg-dark"> <div class="container-fluid"> <a class="navbar-brand" href="#">My Website</a> </div> </nav> <div class="container-fluid"> <div class="row"> <!-- Sidebar: 25% width on medium and up, full width on mobile --> <aside class="col-md-3 bg-light p-4"> <h5>Sidebar</h5> <ul class="list-group"> <li class="list-group-item">Link 1</li> <li class="list-group-item">Link 2</li> <li class="list-group-item">Link 3</li> </ul> </aside> <!-- Main Content: 75% width on medium and up --> <main class="col-md-9 p-4"> <h2>Main Content</h2> <p>This is the main content area. It takes 75% width on tablets and desktops, but 100% on mobile.</p> </main> </div> </div> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script> </body> </html>
Key Points
  • .col-md-3 = 25% (3 out of 12 columns)
  • .col-md-9 = 75% (9 out of 12 columns)
  • On mobile (no class), both take 100% width and stack vertically
  • Use .bg-light to distinguish sidebar visually
  • Perfect for dashboards, documentation sites, blogs
Your Task
  1. Create exercise4-1.html
  2. Build a sidebar layout with:
    • Navigation bar at top
    • Sidebar with navigation menu (use list-group)
    • Main content area with articles/cards
    • Footer
  3. Test responsiveness on mobile (should stack)

Exercise 4.2: Four-Column Equal Layout

Create a four-column layout with equal widths using col-sm-3

Understanding Equal Columns

Bootstrap automatically calculates equal widths:

  • 4 columns × .col-sm-3 (3 cols each) = 12 cols total = perfect fit
  • Each column = 25% width (12 ÷ 4 = 3 columns each)
Code Example
<div class="container mt-5"> <h2 class="mb-4">Four-Column Layout</h2> <div class="row g-3"> <div class="col-sm-3"> <div class="card"> <div class="card-body"> <h5 class="card-title">Feature 1</h5> <p class="card-text">Description here</p> </div> </div> </div> <!-- Repeat 3 more times --> </div> </div>

Responsive Behavior
Screen Size Width Layout
Mobile (< 576px) Full width All 4 columns stack vertically (100% each)
Tablet (≥ 576px) 25% each All 4 columns in one row
Desktop (≥ 768px+) 25% each All 4 columns in one row
Advanced: Auto-Layout Columns

You can also use Bootstrap's auto-layout features:

<!-- Equal width columns (no number needed) --> <div class="row"> <div class="col">Column 1</div> <div class="col">Column 2</div> <div class="col">Column 3</div> <div class="col">Column 4</div> </div>
Your Task
  1. Create exercise4-2.html
  2. Create a four-column card layout with:
    • 4 equal-width cards
    • Each card with image, title, and description
    • Responsive (stack on mobile)
    • Use proper spacing (g-3)
  3. Test on different screen sizes
Column Division Cheat Sheet

2 Columns:

.col-md-6 + .col-md-6

3 Columns:

.col-md-4 + .col-md-4 + .col-md-4

4 Columns:

.col-md-3 × 4

Sidebar (25/75):

.col-md-3 + .col-md-9

Auto Equal:

.col (repeats automatically)
