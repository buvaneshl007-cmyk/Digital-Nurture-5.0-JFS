Exercise 10.1: Responsive Navbar

Create a responsive navbar with logo, links, and search form

Navbar Components
  • .navbar - Container for navigation
  • .navbar-brand - Logo/brand name
  • .navbar-expand-lg - Collapse on mobile, expand on lg+
  • .navbar-dark - Dark theme
  • .navbar-nav - Navigation links list
  • .nav-link - Individual navigation link
Complete Navbar Code
<nav class="navbar navbar-expand-lg navbar-dark bg-dark"> <div class="container-fluid"> <a class="navbar-brand" href="#"> <i class="bi bi-bootstrap-fill"></i> MyBrand </a> <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav"> <span class="navbar-toggler-icon"></span> </button> <div class="collapse navbar-collapse" id="navbarNav"> <ul class="navbar-nav ms-auto"> <li class="nav-item"> <a class="nav-link active" href="#">Home</a> </li> <li class="nav-item"> <a class="nav-link" href="#">About</a> </li> <li class="nav-item dropdown"> <a class="nav-link dropdown-toggle" href="#" data-bs-toggle="dropdown"> Services </a> <ul class="dropdown-menu"> <li><a class="dropdown-item" href="#">Design</a></li> <li><a class="dropdown-item" href="#">Development</a></li> </ul> </li> <li class="nav-item"> <a class="nav-link" href="#">Contact</a> </li> </ul> <form class="d-flex ms-auto"> <input class="form-control me-2" type="search" placeholder="Search"> <button class="btn btn-primary" type="submit">Search</button> </form> </div> </div> </nav>
Key Features
  • Mobile-responsive: hamburger menu on small screens
  • Dropdown menu in navigation
  • Search form aligned right (ms-auto)
  • .navbar-toggler for mobile menu button
  • .navbar-collapse collapsible content
Your Task
  1. Create exercise10-1.html
  2. Build a responsive navbar with:
    • Brand/logo
    • Navigation links
    • Dropdown menu
    • Search form
    • Hamburger menu on mobile
  3. Test on mobile and desktop

Exercise 10.2: Tabs & Pills Navigation

Create tabbed and pill navigation

Nav Tabs

Tabbed navigation for switching between content:

Home content
Profile content
Contact content
Nav Pills

Pill-shaped navigation (alternative to tabs):

Pills Home content
Pills Profile content
Pills Contact content
Tabs Code Example
<ul class="nav nav-tabs" role="tablist"> <li class="nav-item" role="presentation"> <button class="nav-link active" data-bs-toggle="tab" data-bs-target="#tab1">Tab 1</button> </li> <li class="nav-item" role="presentation"> <button class="nav-link" data-bs-toggle="tab" data-bs-target="#tab2">Tab 2</button> </li> </ul> <div class="tab-content"> <div class="tab-pane fade show active" id="tab1" role="tabpanel"> Content for Tab 1 </div> <div class="tab-pane fade" id="tab2" role="tabpanel"> Content for Tab 2 </div> </div>
Your Task
  1. Create exercise10-2.html
  2. Create both nav-tabs and nav-pills:
    • 3+ tabs with content
    • 3+ pills with content
    • Different content for each tab/pill
    • Working tab switching
Navbar Variants
.navbar-dark

Dark navbar with light text

.navbar-light

Light navbar with dark text

.sticky-top

Sticky navbar (stays at top)

.navbar-expand-*

Breakpoint to collapse/expand

Available Breakpoints:
  • .navbar-expand-sm
  • .navbar-expand-md
  • .navbar-expand-lg
  • .navbar-expand-xl
