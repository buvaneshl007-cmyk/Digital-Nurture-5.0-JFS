Exercise 8.1: Registration Form

Create a registration form using form components

Form Components
  • .form-group - Wrapper for form items
  • .form-label - Styled label
  • .form-control - Input/textarea/select styling
  • .form-check - Checkbox/radio styling
  • .input-group - Input with prefix/suffix
Complete Registration Form
<div class="container mt-5"> <div class="row justify-content-center"> <div class="col-md-6"> <h2>Create Account</h2> <form> <div class="mb-3"> <label for="name" class="form-label">Full Name</label> <input type="text" class="form-control" id="name" placeholder="John Doe"> </div> <div class="mb-3"> <label for="email" class="form-label">Email</label> <input type="email" class="form-control" id="email" placeholder="name@example.com"> </div> <div class="mb-3"> <label for="password" class="form-label">Password</label> <input type="password" class="form-control" id="password"> </div> <div class="mb-3"> <label for="country" class="form-label">Country</label> <select class="form-select" id="country"> <option>Select Country</option> <option>USA</option> <option>Canada</option> <option>UK</option> </select> </div> <div class="mb-3 form-check"> <input type="checkbox" class="form-check-input" id="terms"> <label class="form-check-label" for="terms"> I agree to the terms and conditions </label> </div> <button type="submit" class="btn btn-primary w-100">Register</button> </form> </div> </div> </div>

Exercise 8.2: Login Form with Floating Labels

Style a login form using form-floating for modern appearance

What are Floating Labels?

Floating labels are labels that float above the input when focused or filled. This modern design pattern saves space and looks clean.

Floating Labels Code
<form> <div class="form-floating mb-3"> <input type="email" class="form-control" id="floatingEmail" placeholder="name@example.com"> <label for="floatingEmail">Email address</label> </div> <div class="form-floating mb-3"> <input type="password" class="form-control" id="floatingPassword" placeholder="Password"> <label for="floatingPassword">Password</label> </div> <div class="d-grid"> <button class="btn btn-primary btn-lg" type="submit">Sign in</button> </div> </form>
Live Preview

Sign In

Key Points
  • .form-floating wrapper is required
  • Input must come BEFORE label
  • Label moves up when input is focused or has value
  • Great for modern, clean designs
  • Use .d-grid to make button full-width
