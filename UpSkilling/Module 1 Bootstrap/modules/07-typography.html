Class Effect Example

Responsive Alignment
.text-md-center - Center text from medium breakpoint up
.text-lg-end - Right align from large breakpoint up
Common Text Classes Reference
Class Purpose
.text-start Left align (default)
.text-center Center align
.text-end Right align
.text-uppercase Convert to uppercase
.text-lowercase Convert to lowercase
.text-capitalize Capitalize first letter
.text-truncate Truncate with ellipsis
.text-break Break long words
Code Example
<h2 class="text-uppercase text-center">Important Section</h2> <p class="text-capitalize">this text will be capitalized</p> <p class="text-center text-md-end"> This is centered on mobile, right-aligned on larger screens </p>
