Exercise 6.1: Responsive Navbar

Create navbar that stacks vertically on mobile, horizontal on desktop

Flexbox Properties Used
Class Purpose
.d-flex Display: flex (enable flexbox)
.flex-column Stack items vertically
.flex-md-row Arrange horizontally from medium up
.justify-content-between Space items apart
.align-items-center Center items vertically
Complete Navbar Code
<nav class="d-flex flex-column flex-md-row justify-content-between align-items-center p-3 bg-dark text-white"> <h5 class="mb-3 mb-md-0">Brand</h5> <ul class="d-flex flex-column flex-md-row list-unstyled gap-3"> <li><a href="#" class="text-white text-decoration-none">Home</a></li> <li><a href="#" class="text-white text-decoration-none">About</a></li> <li><a href="#" class="text-white text-decoration-none">Contact</a></li> </ul> <input type="text" class="form-control form-control-sm w-100 w-md-25" placeholder="Search"> </nav>
Your Task
  1. Create exercise6-1.html
  2. Build a responsive navbar with:
    • Brand/logo on left
    • Navigation links in middle
    • Search form on right
    • Stacks vertically on mobile
    • Horizontal layout on desktop
  3. Test responsiveness

Exercise 6.2: Card Layout with Flexbox

Use flexbox utilities for proper card spacing and alignment

Card Flexbox Pattern
<div class="card h-100 d-flex flex-column"> <img src="image.jpg" class="card-img-top" alt="..."> <div class="card-body d-flex flex-column flex-grow-1"> <h5 class="card-title">Title</h5> <p class="card-text flex-grow-1">Description text...</p> <a href="#" class="btn btn-primary mt-auto">Learn More</a> </div> </div>
Key Classes Explained
  • .h-100 - Full height of parent
  • .d-flex - Enable flexbox
  • .flex-column - Stack content vertically
  • .flex-grow-1 - Allow content to grow
  • .mt-auto - Push button to bottom
Result: Equal Height Cards with Bottom Buttons

Common Flexbox Utilities
Utility Effect

.flex-grow-1 Grow to fill available space
