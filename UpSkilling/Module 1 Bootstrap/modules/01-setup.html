💻 Exercise 1.1: Quick Start with CDN

The easiest way to get going! No installation, no fuss. Just copy-paste and you're ready. Perfect for beginners and quick projects! 🚀

What You'll Learn
  • How to link Bootstrap CSS and JavaScript via CDN
  • Understanding the meta viewport tag for responsive design
  • Basic HTML structure with Bootstrap
Step-by-Step Instructions
Step 1: Create a new HTML file named exercise1-1.html
Step 2: Copy this basic structure:
<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Bootstrap 5 CDN Setup</title> <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet"> </head> <body> <div class="container mt-5"> <h1>Hello Bootstrap 5!</h1> <p class="lead">This page uses Bootstrap via CDN</p> </div> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script> </body> </html>
Key Points Explained

Meta Viewport Tag: <meta name="viewport" content="width=device-width, initial-scale=1.0"> ensures your page is responsive on mobile devices.

Bootstrap CSS CDN: Links the compiled Bootstrap CSS file from a Content Delivery Network.

Bootstrap JS Bundle: Includes Bootstrap's JavaScript components and their dependencies (Popper.js).

Your Task
  1. Create the HTML file with the provided code
  2. Open it in a browser
  3. Verify that the styling looks different from plain HTML
  4. Try resizing the browser window to see responsive behavior

Exercise 1.2: NPM Setup

Set up a project using npm and downloaded Bootstrap files

What You'll Learn
  • How to set up Bootstrap with npm (no command line needed)
  • Understanding project structure
  • Using downloaded Bootstrap files
Alternative: No-Command Approach (Recommended for Beginners)

Instead of using npm commands, you can download Bootstrap files directly:

Option 1: Download Bootstrap from getbootstrap.com
  1. Visit Bootstrap Download Page
  2. Download the compiled CSS and JS files
  3. Extract the ZIP file in your project folder
  4. Link to the files in your HTML
Sample HTML with Downloaded Files
<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Bootstrap 5 Local Files</title> <link href="bootstrap-5.3.0-dist/css/bootstrap.min.css" rel="stylesheet"> </head> <body> <div class="container mt-5"> <h1>Using Bootstrap from Downloaded Files</h1> <button class="btn btn-primary">Click Me!</button> </div> <script src="bootstrap-5.3.0-dist/js/bootstrap.bundle.min.js"></script> </body> </html>
Folder Structure After Download
your-project/
├── index.html
├── bootstrap-5.3.0-dist/
│ ├── css/
│ │ ├── bootstrap.min.css
│ │ └── bootstrap.min.css.map
│ └── js/
│ ├── bootstrap.bundle.min.js
│ └── bootstrap.bundle.min.js.map
└── style.css (your custom CSS)
Your Task
  1. Download Bootstrap from getbootstrap.com
  2. Extract the files into your project folder
  3. Create an HTML file and link to the local Bootstrap files
  4. Add some Bootstrap components (button, card, etc.)
  5. Open in browser and verify it works
💡 Pro Tip: Use the CDN approach for quick prototyping and learning. Use downloaded files when you need customization or offline development.

Aspect CDN Downloaded Files
