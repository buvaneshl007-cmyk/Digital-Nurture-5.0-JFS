👁️ Module 14: Show & Hide Tricks

Show different content on different devices. Smart responsive design! 📱

Exercise 14.1: Hide/Show Elements

Use display classes to hide/show based on screen size

Display Classes
Class Effect
.d-none Hide element (display: none)
.d-block Display as block
.d-inline Display as inline
.d-flex Display as flex
.d-grid Display as grid
Responsive Visibility
This shows only on medium screens and up
This shows only on mobile
This shows only on large screens
Code Example
<div class="d-none d-md-block"> Shows on medium (md) screens and up </div> <div class="d-block d-lg-none"> Shows on screens smaller than large (lg) </div>
Your Task
  1. Create exercise14-1.html
  2. Create a layout showing:
    • Different content for mobile vs desktop
    • Hidden elements on certain breakpoints
    • Responsive sidebar (show/hide)

Exercise 14.2: Responsive Sidebar

Create a sidebar that only shows on tablets and above

Responsive Sidebar Code
<div class="row"> <aside class="col-lg-3 d-none d-lg-block bg-light p-4"> <h5>Sidebar (desktop only)</h5> <ul class="list-group"> <li class="list-group-item">Link 1</li> <li class="list-group-item">Link 2</li> </ul> </aside> <main class="col-lg-9"> <h2>Main Content</h2> </main> </div>
Your Task
  1. Create exercise14-2.html
  2. Build a responsive layout with:
    • Sidebar visible only on lg and up
    • Mobile-optimized main content
    • Proper responsive behavior
Breakpoint Reference

.d-none - Always hide

.d-sm-block - Show from sm up

.d-md-block - Show from md up

.d-lg-block - Show from lg up

.d-xl-block - Show from xl up

.d-md-none - Hide from md up
